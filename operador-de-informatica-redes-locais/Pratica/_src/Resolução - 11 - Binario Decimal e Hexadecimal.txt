3. Aritmética Binária 
---------------------

Resolva as seguintes operações de soma:

.. __1 0 1 + 1 0__ = 1 1 1
.. __1 1 1 + 1 0__ = 1 0 0 1
.. __1 0 1 0 + 1 1 0__ = 1 0 0 0
.. __1 0 1 0 + 1 0 1 0__ =1 0 1 0 0
.. __1 0 1 0 + 1 1 1 1__ =1 1 0 0 1
.. __1 0 1 1 + 1 0 1__ = 1 0 0
.. __1 1 1 1 0 + 1 1 1 0__ = 1 0 1 1 0 0
.. __1 0 0 1 1 0 + 1 1 0 1__ = 1 1 0 0 1 1
.. __1 0 1 1 1 0 + 1 0 1__ = 1 1 0 0 1 1
.. __1 0 1 + 1 1__ = 1 0 0 0
.. __1 1 1 + 1 1 1__ = 1 1 1 0
.. __1 0 1 0 + 1 0 1 0__ = 1 0 1 0 0
.. __1 1 1 0 1 + 1 0 1 0__ = 1 0 0 1 1 1
.. __1 1 1 1 1 + 1 1 1 1 1__ = 1 1 1 1 1 0
