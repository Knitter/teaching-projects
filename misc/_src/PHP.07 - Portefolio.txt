Portefólio
==========

Todos os programadores devem guardar os trabalhos que desenvolvem de modo a construir um portefólio completo e que demonstre a evolução e as capacidades adquiridas. Um portefólio deverá mostrar a quem o visualiza quais os trabalhos feitos pelo programador mas, além de mostrar, será importante que o portefólio apresente interactividade e permita experimentar alguns dos trabalhos desenvolvidos.

Com o objectivo de criar uma montra do que é desenvolvido, pretende-se o desenvolvimento de um portefólio Web, com alguma interatividade e que contenha, pelo menos, uma listagem de todos os projectos PHP, JavaScript ou HTML que vão sendo desenvolvidos.

O portefólio deverá ser contituído por *três* páginas de texto: a página inicial com um _slideshow_ de _screenshots_ dos projectos, uma página de contactos com um formulário funcional e uma página com a descrição do portefólio.

Além destas páginas, deverá existir uma página por cada projeto com o nome do projeto, a descrição e o projeto ou uma imagem do mesmo se não for possível incluir o projeto na página que lhe está destina. Neste último caso deverá existir uma hiper-ligação para o local onde o projeto possa ser experimentado.
